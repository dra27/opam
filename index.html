OPAM on Native Windows

Version af03a14

SHA512SUMS for all four zip files.

  • mingw: opam-af03a14-mingw.zip
  • mingw64: opam-af03a14-mingw64.zip
  • msvc: opam-af03a14-msvc.zip
  • msvc64: opam-af03a14-msvc64.zip

Each zip contains OPAM compiled with a different flavour of OCaml 4.02.3. The choice of which zip to use does not affect the switches available: on 64-bit Windows, all four versions of OPAM can maintain 32- and 64-bit switches. Obviously, to run either a 64-bit build of OPAM or install a 64-bit switch, you must actually be using 64-bit Windows…
